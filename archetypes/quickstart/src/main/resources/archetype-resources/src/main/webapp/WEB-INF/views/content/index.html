REST support?

Sure thing! Check the WEB-INF/config/routes.yml for some examples, and edit them or add new ones. The routes.yml file is monitored by Micro's RouteManager and the changes are available right away without needing to restart the app; this is only happening if MICRO_ENV is: development.

Use curl and run the following command line: 

    curl http://localhost:8080/echo/John
or see this browser session and follow the next link: 
    - route: /echo/{name}

Learn more about the REST support: Micro and REST
